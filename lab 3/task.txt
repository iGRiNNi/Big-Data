Задание на Итоговую работу по numpy, pandas, matplotlib
Работа с признаками. Очистка данных: обработка пропусков и выбросов. Построение диаграмм.
Работу необходимо выполнять в Jupyter Notebook. Ответом на данное задание будет являться файл с расширением ipynb, который содержит итоговый код. Каждый пункт задания необходимо выполнять в одной или нескольких отдельных ячейках. 
Использовать можно только модули numpy, pandas, matplotlib, прочие модули использовать запрещено. 
Разработать программу, которая реализует предварительную обработку данных. В качестве входных данных используется статистика по покупателям молла (модифицированный набор данных https://www.kaggle.com/shwetabh123/mall-customers, файл mall_customers.csv).


Задание 4
1. Считать данные из файла и вывести метод первые 20 его строк (метод head()). 
2. Проверить наличие пропусков в файле. Если пропуски найдены, то заменить их на медианное значение во всех числовых столбцах. 
3. Вычислить количество мужчин и женщин до 50 лет. И для каждого пола определить суммарные расходы (столбец Spending Score). 
4. Оценить корреляцию Пирсона для столбцов Age и Annual Income. 
5. Сохранить в новый файл (output.csv) данные по покупателям, у которых доход (Annual Income) <= 70. 
6. Для данных о покупателях построить круговую диаграмму по распределению возрастов (по интервалу в 10 лет). 
7. Построить столбчатую диаграмму по распределению доходов (по интервалу 25 тысяч).
